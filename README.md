# bazzite-ai-notebooks

Example Jupyter notebooks for [bazzite-ai](https://github.com/atrawog/bazzite-ai).

## Notebooks

| Notebook | Description |
|----------|-------------|
| `01-cuda-verification.ipynb` | Verify CUDA/GPU access in Jupyter environment |
| `02-pytorch-quickstart.ipynb` | PyTorch quickstart tutorial with GPU support |
| `03-data-science-template.ipynb` | Full data science workflow template |

## Usage

These notebooks are designed for the bazzite-ai Jupyter pod environment with:
- Python 3.13
- PyTorch with CUDA 12.6
- Full ML/AI stack (transformers, langchain, HuggingFace ecosystem)

Start the Jupyter environment:
```bash
ujust jupyter start
```
